Karakter ini adalah seekor babi lucu bernama "Si lucu" wajah babi digambar menggunakan bentuk lingkaran, 
berwarna pink mudah sebagai kepala, dua telinga segitiga, dan hidung oval dengan dua lubang hidung kecil.
Memiliki dua mata bulat hitam dan senyuman kecil agar terlihat ramah dan imut.
Teks " si lucu" ditambahkan dibagian bawa untuk memberi nama pada karakter.

Karakter ini dibuat dengan fungsi-fungsi dasar dari OpenCV seperti:
1. cv2.circle () untuk kepala, mata, dan hidung,
2. cv2.ellipse () untuk mulut dan hidung,
3. cv2.fillpoly () untuk telinga,
4. cv2. putText () untuk teks label nama.
